Python-скрипт (пока не экзешник) для интеграции с [PreMiD](//premid.app/) (пока только с ним), отображающий ваш статус в профиле Telegram во вкладке сохранённой музыки.

## Установка и использование
Пока никак. Простите.

## Разработка
- Установите библиотеки из `requirements.txt`
- ...ну и всё.
### Схема проекта:
- `server.py` - интеграция с PreMiD и его системой Activity Forwarding, который требует запуска веб-сервера
- `tags.py` - работа с тегами аудиофайла, приложенного в репозиторий (сам файл не нужно редактировать)
- `config.py` - на данный момент костыль, планируется переработать в более удобной графической форме
- `client.py` - управление Telegram аккаунтом, требует регистрации в [панели разработчика](//my.telegram.org) (по хорошему для билдов надо предоставлять свои ключи чтоб пользователям не приходилось вводить свои, но я не хочу рисковать своими ключами чтобы попытаться поиграть в обфускацию)
- `interface.py` (и папка `interface`) - графический интерфейс под управлением Qt6
### Проверочный запуск
Первый запуск `main.py` (или `config.py`, если вы такой упрямый) сгенерирует базовую конфигурацию. Внесите туда свои данные и перезапустите файл.
