Python-скрипт для интеграции с [PreMiD](https://premid.app/), отображающий ваш статус в профиле Telegram во вкладке сохранённой музыки.

## Установка и использование
- Установите библиотеки Telethon, mutagen, aiohttp и qrcode
- Зарегистрируйтесь на [my.telegram.org](https://my.telegram.org)
- Запустите main.py, сгенерируется файл конфига
- Введите api_id и api_hash в файл конфига (ни с кем ими не делитесь!) ((есть ещё другие настройки, но они необязательны))
- Установите PreMiD и поставьте в настройках Forwarding URL `http://localhost:1225/trp/premid` (или тот порт что вы указали в конфиге)
- Запустите main.py

После запуска скрипта ваши просматриваемые ролики / прослушиваемая музыка будет отображаться в профиле как аудиотрек. (засуну сюда картинку потом)
